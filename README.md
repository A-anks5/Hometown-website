# Visit Bengaluru

A beautifully designed travel guide website showcasing the best attractions and experiences in Bengaluru, India.

## 📋 Project Overview

"Visit Bengaluru" is a responsive web project that introduces visitors to the vibrant city of Bengaluru through an elegant, user-friendly interface. The site highlights three must-do activities and features a local guide to help travelers make the most of their visit.

## 🎯 Features

- **Hero Section**: Eye-catching header with the Vidhan Soudha image and tagline
- **Top Activities Section**: Three featured attractions with descriptions:
  - Sunrise at Nandi Hills
  - Cafés in Koramangala
  - Nightlife of Church Street
- **Guide Profile Card**: Information about your local guide, Izzy Agreste
- **Responsive Design**: Clean, centered layout with elegant typography
- **Interactive Elements**: Hover effects on the guide card and external links to attractions

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Styling with flexbox layout
- **Google Fonts**: Custom typography
  - Sansita Swashed (titles)
  - Lobster Two (subtitles)
  - Playfair Display (body text and headers)
  - Montserrat Alternates (guide information)

## 🎨 Design Highlights

- **Color Scheme**: 
  - Primary: #5a613b (olive green)
  - Secondary: #a5b4c8 (muted blue)
  - Background: #eef4ed (soft sage)
  - Card: #F8F1E7 (warm cream)

- **Typography**: Mix of serif and sans-serif fonts for visual hierarchy
- **Layout**: Flexbox-based responsive design
- **Visual Effects**: Rounded corners, text shadows, and hover animations

## 📖 Usage

1. Open `visithometown.html` in a web browser
2. Navigate through the different sections to explore activities
3. Click on activity links to learn more from external sources
4. View guide information in the profile card
